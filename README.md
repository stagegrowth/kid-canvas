# Kid Canvas 🎨

5살 자녀를 위한 색칠 놀이 안드로이드 앱.

## 구성
- `app/` — 안드로이드 앱 (Kotlin + Jetpack Compose)
- `tools/content-builder/` — 콘텐츠 빌드 도구 (Python)
- `docs/` — 개발 플랜 및 Claude Code 프롬프트
- `inputs/` — 콘텐츠 원본 (PDF/PNG, gitignore 처리됨)
